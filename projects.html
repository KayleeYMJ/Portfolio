UQ Easy Go

This is a mini-program of WeChat developed by "kexing" organization, which allows users to comment courses, set deadlines of assignments, set timetable, calculate final marks and so on. We used some technicals to develop it, such as crawler for fetching the courses' information, databases for storing users' information, Github for version control and so on.

I mainly took responsible for the "course commenting" function by using React.js and JavaScript, such as translating UI/UX designs into responsive and visually appealing interfaces, as a front-end developer. Additionally, there are four icons designed by me, which are on the home page, including "calculator" icon, "countdown" icon, "timetable" icon and "course commenting" icon.

Find Koala

A website designed for travel recording and experience sharing, which are used to record people's travel photos and videos in real time, and also it automatically generates "travel memories", which can be used for people to enjoy and share with their friends. Additionally, this app allows users to export their experience in different formats, such as a zip folder.

As a full-stack developer, I handled with translating UI designs into responsive and visually appealing interfaces and also designed databases, based on Uniapp. Technicals mainly include Vue.js, mysql, JavaScript and CSS.

Millionaire Assistant

This website aims to assist food merchants in Brisbane who want to boost their income by putting food trucks. There is some information about enrolled food trucks’ geographical distributions, the type of food, the locations of bus stops and ferry terminals, the recommended suburbs and other details for merchants to make a decision about where is better to put a food truck.

Functions include filtering, searching, data analysis, recommended locations and so on. I took responsible for achieving filtering function and showing some details of a selected food truck. For example, all locations would be shown on the map, after filtering burgers.

End Of Dayz

This is a small game which allows zombies to pursue a player. If zombies run into a player, the player would be lose. When the player picks up an item, that item is added to an inventory, but not immediately applied. The inventory is displayed to the right of the game map. The user can activate and deactivate items by left clicking on them in the inventory. When an item is activated, it begins working, and its lifetime decreases by 1 every second. Additionally, if the player arrive at the hospital, the player would be win.
